# Cron Templates

These files are prompt templates, not mandatory live jobs.

## Recommended posture

- start advisory-first
- keep outputs concise
- avoid automatic mass writeback
- do not enable auto-synthesis by default

## Included templates

- `inbox-triage.md`
- `wiki-lint.md`
- `weekly-structural-digest.md`

## Typical cadence

- Inbox triage: 1-3 times daily
- Wiki lint: daily or every evening
- Structural digest: weekly
