# Wiki Lint Prompt

Audit `WIKI_ROOT` for structural health.

## Required checks
- navigation health
- duplicates / synonyms
- orphan or weakly linked pages
- stale core pages
- missing high-value pages
- backlog candidates in `INBOX_ROOT`

## Output requirements
- concise report
- P1 / P2 / P3 structure
- audit-only by default
- no silent file mutations
